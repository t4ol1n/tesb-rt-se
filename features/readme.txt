List of TESB features:

tesb-sam-agent
--------------
install the Agent to the OSGi Container.

tesb-derby-starter
------------------
install the Derby database into the OSGI container and start running it.

tesb-sam-server
---------------
install Monitoring Server in a Talend ESB container:

tesb-locator-client
-------------------
install locator client in a Talend ESB container

tesb-locator-soap-service
-------------------------
install locator soap service in a Talend ESB container

tesb-locator-rest-service
-------------------------
install locator rest service in a Talend ESB container

tesb-zookeeper-server
---------------------
install and run zookeeper server in a Talend ESB container.

talend-job-api
---------------------
install Talend Job API in a Talend ESB container.

talend-job-controller
---------------------
install and run Job Controller in a Talend ESB container.
